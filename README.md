## 🚀 The Vision (V5 Premium Release)
The **Andhra Pradesh Road Safety AI** is a world-class, full-stack SaaS platform designed to revolutionize highway maintenance and driver safety. By merging **YOLOv8 AI**, **High-Fidelity 3D Mapping**, and **Automated Government Routing**, we provide a professional ecosystem for both citizens and road safety officials.

---

## ✨ 🆕 V5 Premium Features

### 🤖 1. AI-Powered Road Audit Engine
- **Full-Speed Detection**: Identification of potholes and road hazards at 30 FPS using custom-trained **YOLOv8** (`best.pt`).
- **Automated Repair Billing**: Itemized cost estimation (INR) based on pothoel dimensions and severity.
- **Official Export**: Generates government-compliant PDF inspection logs instantly.

### 🗾 2. 3D Safety Navigator (GIS)
- **High-Fidelity Maps**: Uses **Folium** and **ESRI Satellite** imagery for 3D road visualization.
- **Safe Path Planning**: Enter your route and see pulsing **Red Hazards** before you drive.
- **Blackspot Management**: Identifies high-accident zones with pulsing skull markers to warn drivers.

### 🏛️ 3. Premium Authority Portal
- **Glassmorphism UI**: Futuristic, transparent dashboard with glowing neon metrics.
- **District Routing**: Automated reporting to any of the **26 Andhra Pradesh District Offices** via Brevo API.
- **Official Footer**: A multi-tier, corporate-grade footer with developer links and office info.

### 📍 4. Smart GPS Location Engine
- **Pinpoint Browser GPS**: Captured instantly during live scans.
- **Smart Guess Logic**: Automatically geocodes road locations for videos uploaded from different locations later.
- **Google Maps Integration**: Direct links to view pothole positions on Google Maps for immediate repair dispatch.

## 🛠️ Quick Start

1. **Clone & Enter**:
   ```bash
   git clone https://github.com/ravindra-gogineni/road-audit-ai.git
   cd road-audit-ai
   ```

2. **Install Specs**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch Dashboard**:
   ```bash
   streamlit run app.py
   ```
